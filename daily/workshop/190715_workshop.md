# 1. 문제 1번

* 두 개의 정수 n과 m이 주어집니다. **반복문을 사용하지 않고** 별(*) 문자를 이용해 가로의 길이가 n, 세로의 길이가 m인 직사각형 형태를 출력해보세요.

```python
# 문제
n = 5
m = 9

# 답
print(f"{'*' * n}\n" * m)
```

```
# 결과
*****
*****
*****
*****
*****
*****
*****
*****
*****
```

* `*` 를 n번 출력하고 마지막에 개행문자인 `\n`를 넣은 후 그 문장을 다시 m번 출력하면 직사각형이 출력된다.
